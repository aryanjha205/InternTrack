Create a modern Internship Attendance Calculator PWA using Python (Flask or FastAPI), HTML, CSS, and JavaScript that calculates internship attendance from 1 January 2026 to 30 April 2026 according to BISAG-N internship rules. The system should automatically detect working days, Sundays, and 2nd & 4th Saturdays as weekly holidays, and also exclude official Gujarat government holidays such as Makar Sankranti (14 Jan 2026), Republic Day (26 Jan 2026), Dhuleti/Holi (4 Mar 2026), Cheti Chand (19 Mar 2026), Id-ul-Fitr (21 Mar 2026), Ram Navami (26 Mar 2026), Mahavir Jayanti (31 Mar 2026), Good Friday (3 Apr 2026), and Dr. B. R. Ambedkar Jayanti (14 Apr 2026) while calculating attendance. The application should allow users to enter attendance day-by-day (Present/Leave), upload leave approval PDF for verification, and automatically calculate total working days, holidays, leave days, present days, and attendance percentage. The system must display Eligibility Status based on 80% attendance criteria, where green indicator shows “Eligible for Internship Completion” and red indicator shows “Not Eligible – Attendance below 80%” along with a warning notice and required remaining attendance information. Include features such as interactive calendar view with holiday markers, automatic detection of 2nd and 4th Saturdays, attendance progress bar, analytics charts, and a summary dashboard. The UI should be very attractive with gradient colors, glassmorphism cards, smooth animations, and responsive mobile-friendly design. Convert the application into a Progressive Web App (PWA) with offline support, service worker, install button, push notification reminders for attendance entry, and a custom app icon, so interns can easily track their daily attendance and eligibility status in a visually engaging way.
